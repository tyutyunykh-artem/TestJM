# TestGame
2D-игра для тестового задания

# Версия Unity
Unity 2022.3.62f1 (LTS)

# Стек
DI-контейнер - VContainer  
Async/Await - UniTask  
Реактивность - R3  
Реактивные коллекции - ObservableCollections  
UI - uGUI + TMP  
Анимации - DOTween  
Сериализация - Newtonsoft.Json  
Локализация - Unity Localization
